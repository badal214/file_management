# File Distribution Portal

A web-based document distribution system designed to manage and share digital files efficiently, reducing the dependency on physical document handling.

---

## 🚀 Features

- Upload and manage digital documents
- Secure file retrieval from database
- Organized document storage
- Responsive user interface
- Backend integration with database

---

## 🛠 Tech Stack

Frontend  
- HTML5  
- CSS3  
- JavaScript  

Backend  
- PHP  

Database  
- MySQL  

--
## ⚙️ How to Run the Project

1. Install **XAMPP / WAMP**
2. Move the project folder to the **htdocs** directory
3. Start **Apache and MySQL**
4. Import the database into **phpMyAdmin**
5. Open the project in browser

## 📚 Learning Outcomes

- Implemented backend communication using PHP and MySQL
- Designed responsive UI using HTML, CSS, and JavaScript
- Structured database for document storage and retrieval
